Project DM

Status: v.1 Complete

This project was completed in just two weeks using off the shelf components from a variety of manufacturers. The goal was to create what a HTPC should be, something that was extremely easy to use with a 10-foot UI. This means NO mouse or full-size keyboard.

The device can be turned on/off and controlled using an iPhone or an iPod Touch. It currently runs a custom build of Windows Embedded Standard 7. Future versions will drop Windows and I will focus on custom builds of Linux. The purpose of this project is to create an easy to use 10-foot UI but is also customizable for technical users such as myself. I also plan to design and build an enclosure made entirely out of aluminum through CNC machine processes.

Project DM device

Details

  • Dual Core Atom 330 overclocked to 2GHz, 4GB RAM, 1.2TB Storage
  • 24/7/365 Media Server with a power requirement of only 25 watts!
  • Fully networked (WiFi/Wired) and controlled from any remote location.
  • Play any video and audio format: MKV, MP4, FLAC, AVI, XVID, VC1, x264, H.264/AVC, etc.
  • Full 1080P HDMI video output.

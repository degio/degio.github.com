Project XRT

Status: In Progress

I want to create a new web GUI that will reinterpret the way we view pictures on the internet. I will be working with the Instagr.am API for the content. Fully utilizing HTML5 and CSS3, I want to take a different approach to the way pictures are both presented and organized.

The image on the right is that of starburst galaxy M82. It was created using images from different telescopes, including the Chandra X-ray Observatory (shown in blue). The combination of these images presents a view from a multiple of perspectives. This is exactly what I want to do with this project. Take different perspectives of UX/UI picture viewing and create an entirely different experience.

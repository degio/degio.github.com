To see the difference between static and fixed top navbars, just scroll.

Imagery

These are the image dimensions and styles that should be used throughout the site. All images will be responsive. The minimum image size should be 1200px wide if you want to maintain a respectable level of quality. Althought there might not be a section that will display images at the full 12 column width, it is certainly possible. Images should maintain a 4:3 ratio. So the dimensions should be 1200x900px.
